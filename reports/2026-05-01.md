## 1. ASR (End-to-End, Diarization, Whisper)

### **Sortformer: A Novel Approach for Permutation-Resolved Speaker Supervision in Speech-to-Text Systems (Taejin Park, Ivan Medennikov, Kunal Dhawan, Weiqing Wang, He Huang, Nithin Rao Koluguri, Krishna C. Puvvada, Jagadeesh Balam, Boris Ginsburg — NVIDIA, 2024-09)** — ICML 2025 accepted
- arxiv URL: https://arxiv.org/abs/2409.06656
- 기존 문제: End-to-end 다화자 ASR에서 화자 permutation 문제를 해결하기 위해 permutation invariant loss(PIL)에만 의존하면 학습 불안정성이 존재하고, 기존 diarization과 ASR이 별도 파이프라인으로 분리되어 있어 통합이 어려움.
- 기존 방법: PIL 기반 EEND(End-to-End Neural Diarization), SOT(Serialized Output Training) 기반 다화자 ASR, 별도의 diarization 후 ASR 결합 파이프라인.
- 제안 방법: Sort Loss라는 새로운 학습 목표를 도입하여 diarization 모델이 자율적으로 화자 순열을 해결하도록 함. PIL 없이 또는 PIL과 결합하여 사용 가능. 또한 sinusoidal kernel function을 이용해 Sortformer의 화자 라벨 추정을 ASR encoder state에 직접 임베딩하여 diarization과 ASR을 seamless하게 통합.
- 핵심 차별점: Sort Loss가 permutation 문제를 정렬 기반으로 해결하여 PIL의 한계를 보완하고, 화자 timestamp와 ASR token을 직접 연결(bridging)하는 구조로 단일 모델에서 diarization+ASR 동시 수행.
- 실험 결과: Sort Loss와 PIL 결합 시 기존 PIL 전용 SOTA end-to-end diarization 모델과 경쟁적 성능 달성. Sortformer의 화자 supervision 통합이 다화자 전사 정확도를 개선. 코드 및 모델이 NVIDIA NeMo 프레임워크를 통해 공개.
- 우리 연구 관련도: ★☆☆ (화자 permutation 해결 메커니즘 및 encoder 내 화자 정보 임베딩 설계는 참고할 만하지만, VQ/SSL 표현 학습과는 직접 관련 낮음)

### **DM-ASR: Diarization-aware Multi-speaker ASR with Large Language Models (Li Li, Ming Cheng, Weixin Zhu, Yannan Wang, Juan Liu, Ming Li, 2026-04)** — arxiv preprint
- arxiv URL: https://arxiv.org/abs/2604.22467
- 기존 문제: 다화자 ASR은 "누가 무엇을 언제 말했는가"를 동시에 모델링해야 하지만, 화자 귀속·시간 구조·어휘 인식을 joint로 학습하는 것은 어렵고 대규모 데이터가 필요. 기존 Speech-LLM 기반 통합 접근법도 이 어려움에서 자유롭지 않음.
- 기존 방법: SOT 기반 end-to-end 다화자 ASR, separate diarization+ASR 파이프라인, 최근 Speech-LLM 기반 unified 모델.
- 제안 방법: 다화자 ASR 과제를 multi-turn dialogue generation 프로세스로 재정의. 오디오 청크와 diarization 결과를 입력받아 각 화자·시간 세그먼트에 대응하는 speaker- and time-conditioned query 시퀀스로 분해하여 LLM이 세그먼트별 전사를 생성.
- 핵심 차별점: Reliable diarization을 명시적 structural prior로 활용하여 joint 학습의 복잡도를 줄이고, 대화 생성 프레임워크로 재정의함으로써 소규모 모델과 적은 학습 데이터로도 강건한 성능을 달성.
- 실험 결과: 중국어(Mandarin) 및 영어 벤치마크에서 비교적 작은 모델과 학습 데이터로 강력한 성능을 보이며, 기존 unified 접근법과 경쟁적이거나 우수한 결과 달성.
- 우리 연구 관련도: ★☆☆ (LLM 기반 ASR 아키텍처 설계 중심으로 SSL/VQ 표현 연구와 직접 관련성은 낮음)

### **Whisper-LM: Improving ASR Models with Language Models for Low-Resource Languages (Xabier de Zuazo, Eva Navas, Ibon Saratxaga, Inma Hernáez Rioja — HiTZ Center, UPV/EHU, 2025-03)** — arxiv preprint
- arxiv URL: https://arxiv.org/abs/2503.23542
- 기존 문제: Whisper와 같은 대규모 다국어 ASR 모델이 저자원 언어에서 해당 언어의 고유한 언어학적 특성을 충분히 반영하지 못해 성능이 크게 저하됨. 특히 domain shift가 발생하면 성능 감소가 심각.
- 기존 방법: Whisper 파인튜닝, 외부 n-gram LM을 beam search에 결합하는 shallow fusion, 기존 통계적 LM 접근.
- 제안 방법: 파인튜닝된 Whisper 모델에 전통적 통계 LM(n-gram)과 대규모 LLM을 추론 시점에서 통합(score merging). 외부 언어 prior를 Whisper의 내부 스코어와 병합하여 domain-shift 문제를 완화.
- 핵심 차별점: 전통적 n-gram과 최신 LLM을 모두 Whisper에 통합하여 비교 분석한 첫 체계적 연구. n-gram이 in-distribution에서, LLM이 다양한 언어 맥락에서 각각 강점을 보임을 실증.
- 실험 결과: 통계 LM 적용 시 in-distribution 데이터셋에서 최대 51% WER 감소, out-of-distribution에서 최대 34% WER 감소. LLM은 다양한 언어 맥락에서 moderate하지만 일관된 개선 제공.
- 우리 연구 관련도: ★☆☆ (언어 모델 결합 기반 ASR 개선으로, speech representation/SSL 연구와 직접 관련성은 낮음)

## 2. SpeechLLM (Voice Agent, Spoken Dialogue, Tool Use)

### **Audio2Tool: Bridging Spoken Language Understanding and Function Calling (Ramit Pahwa, Apoorva Beedu, Parivesh Priye, Rutu Gandhi, Saloni Takawale, Aruna Baijal, Zengli Yang, 2026-04)** — arxiv preprint
- arxiv URL: https://arxiv.org/abs/2604.22821
- 기존 문제: SpeechLM이 대화와 이해 능력에서 발전했지만, 실세계 음성 기반 도구 호출(function calling) 능력을 체계적으로 평가할 벤치마크가 부재. 기존 벤치마크는 텍스트 기반이거나 단순한 음성 이해에 한정.
- 기존 방법: 텍스트 기반 LLM function calling 벤치마크, 단순 음성 명령 인식 시스템, ASR→LLM 캐스케이드 파이프라인 기반 도구 호출.
- 제안 방법: Smart Car, Smart Home, Wearables 3개 도메인에 걸쳐 약 30,000개 쿼리로 구성된 대규모 벤치마크. 단순 직접 명령부터 복합 다중 의도(multi-intent), needle-in-a-haystack 추출까지 다층 복잡도 계층을 설계. Zero-shot voice cloning TTS와 다양한 노이즈 프로파일로 현실적 음향 조건을 시뮬레이션.
- 핵심 차별점: SpeechLM의 tool-calling 능력을 체계적으로 평가하는 최초의 대규모 음성 벤치마크. 복합 의도 및 음향 도전 시나리오에서 현재 시스템의 한계를 정량화.
- 실험 결과: SOTA SpeechLM 및 ASR-LLM 파이프라인 모두 단순 명령에서는 강한 성능을 보이지만, 복합 의도(compositional) 및 음향 도전(acoustic challenge) 조건에서 현저한 성능 저하 확인.
- 우리 연구 관련도: ★☆☆ (벤치마크/평가 중심 기여로 SSL/VQ 표현 연구와 직접 관련성은 낮음)

### **VoxMind: An End-to-End Agentic Spoken Dialogue System (Tianle Liang, Yifu Chen, Shengpeng Ji, Yijun Chen, Zhiyang Jia, Jingyu Lu, Fan Zhuo, Xueyi Pu, Yangzhuo Li, Zhou Zhao — Zhejiang University et al., 2026-04)** — arxiv preprint
- arxiv URL: https://arxiv.org/abs/2604.15710
- 기존 문제: 사용자 요구가 복잡해지면서 대화 능력만으로는 한계가 있고, 도구 사용(tool use)을 통해 지식 경계를 확장하는 agentic 능력이 필수적이지만, 기존 end-to-end spoken dialogue 모델은 이러한 agentic 능력이 부족.
- 기존 방법: ASR→LLM→TTS 캐스케이드 파이프라인, 또는 end-to-end 음성 대화 모델(대화만 가능, 도구 호출 불가).
- 제안 방법: End-to-end spoken dialogue 모델에 포괄적 agentic 능력을 부여하는 VoxMind 프레임워크. (1) 470시간 규모 AgentChat 데이터셋 구축, (2) "Think-before-Speak" 메커니즘으로 구조화된 추론을 모델에 내재화, (3) Multi-Agent Dynamic Tool Management로 보조 에이전트에게 검색 태스크를 비동기 위임하여 추론 지연을 toolset 크기와 분리.
- 핵심 차별점: End-to-end spoken dialogue 모델에 agentic tool-use 능력을 통합한 최초의 프레임워크. 추론-발화 분리와 다중 에이전트 관리로 지연 문제를 해결하면서 대화 품질을 유지.
- 실험 결과: Task completion rate이 34.88%에서 74.57%로 대폭 향상. Spoken agent 과제에서 Gemini-2.5-Pro를 능가하면서도 일반 대화 품질 유지. 코드 및 데이터 공개.
- 우리 연구 관련도: ★☆☆ (agentic 대화 시스템 설계 중심이며 speech representation 학습과는 거리가 있음)

### **SLAM-LLM: A Modular, Open-Source Multimodal Large Language Model Framework and Best Practice for Speech, Language, Audio and Music Processing (Ma Ziyang, Yang Guanrou, Chen Wenxi, Gao Zhifu et al. — Shanghai Jiao Tong University, X-LANCE Lab, 2026-01)** — IEEE JSTSP accepted
- arxiv URL: https://arxiv.org/abs/2601.09385
- 기존 문제: 대부분의 MLLM 프레임워크가 vision을 주 입력 모달리티로 삼아, speech/audio/music 모달리티에 대한 심층 지원이 부족. 이로 인해 연구자들이 코드 작성과 하이퍼파라미터 튜닝에 많은 노력을 소모.
- 기존 방법: 범용 MLLM 프레임워크(LLaVA 등)에서 음성을 부가 모달리티로 추가하는 방식, 또는 개별 연구마다 custom 코드를 작성하는 방식.
- 제안 방법: Speech, Language, Audio, Music 처리를 위한 모듈화된 오픈소스 MLLM 프레임워크. 다양한 encoder, projector, LLM backbone, PEFT 플러그인을 모듈식으로 조합 가능. ASR, Automated Audio Captioning(AAC), Music Captioning(MC) 등 주요 태스크별 training/inference 레시피와 고성능 체크포인트를 제공.
- 핵심 차별점: Vision 중심 MLLM 생태계에서 speech/audio/music을 일급 모달리티로 지원하는 최초의 체계적 프레임워크. 모듈식 설계로 encoder-projector-LLM 조합을 자유롭게 실험 가능.
- 실험 결과: 제공된 레시피 중 일부가 ASR, AAC, MC 등에서 SOTA 수준에 근접하거나 달성. 관련 기법이 별도 학술 논문으로도 채택됨.
- 우리 연구 관련도: ★★☆ (다양한 speech encoder(SSL 포함)를 LLM과 결합하는 프레임워크로, SSL 표현의 downstream 활용 실험 환경으로 직접 유용)

## 3. Speech Foundation Model (SSL, Tokenizer, Representation)

### **SiTok: Scaling Speech Tokenizers with Diffusion Autoencoders (Yuancheng Wang, Zhenyu Tang, Yun Wang, Arthur Hinsvark, Yingru Liu, Yinghao Aaron Li, Kainan Peng, Junyi Ao, Mingbo Ma, Mike Seltzer, Qing He, Xubo Liu — CUHK-Shenzhen, Microsoft et al., 2026-02)** — arxiv preprint
- arxiv URL: https://arxiv.org/abs/2602.06602
- 기존 문제: Speech tokenizer는 의미(semantics) 인코딩과 음향(acoustics) 재구성 사이의 trade-off 문제를 안고 있으며, 낮은 bit rate와 낮은 token rate를 동시에 달성하기 어려움.
- 기존 방법: EnCodec, SpeechTokenizer 등 RVQ 기반 neural codec, HuBERT k-means 기반 이산화, 별도의 semantic/acoustic 분리 tokenizer.
- 제안 방법: Diffusion autoencoder 기반 SiTok. (1) 지도 학습을 통한 semantic-rich representation 학습과 diffusion을 통한 고충실도 음성 재구성을 동시 수행, (2) semantic regularization으로 이해 과제 성능 강화, (3) shortcut fine-tuning으로 diffusion 디코딩을 대폭 가속. 1.6B 파라미터, 22M 시간(2200만 시간) 음성으로 학습.
- 핵심 차별점: Diffusion decoder를 tokenizer에 도입하여 극저 token rate(12.5 Hz, 200 bps)에서도 semantics와 acoustics를 동시에 유지. 대규모 스케일링(1.6B 파라미터)을 통해 이해·재구성·생성 3개 축에서 기존 baseline을 동시에 초월.
- 실험 결과: 12.5 Hz token rate, 200 bps의 극저 bit rate에서 이해(speech understanding), 재구성(reconstruction quality — WER, speaker similarity, UTMOS), 생성(speech generation) 벤치마크 모두에서 강력한 baseline들을 능가.
- 우리 연구 관련도: ★★★ (VQ/이산 표현 + semantics-acoustics 동시 모델링이 VQ-based speech representation 연구와 직결. Diffusion 기반 디코더 설계는 prediction-free representation의 대안적 생성 경로로 참고 가치 높음)

### **WavTokenizer: an Efficient Acoustic Discrete Codec Tokenizer for Audio Language Modeling (Shengpeng Ji, Ziyue Jiang, Wen Wang et al. — Zhejiang University, Alibaba, Meta FAIR, 2024-08)** — ICLR 2025 accepted
- arxiv URL: https://arxiv.org/abs/2408.16532
- 기존 문제: 기존 neural audio codec(EnCodec 등)은 다층 RVQ를 사용하여 토큰 시퀀스가 길어지고, Audio LM의 autoregressive 모델링에서 시퀀스 길이와 복잡도가 급증. 단일 코드북으로 압축하면 재구성 품질이 크게 저하됨.
- 기존 방법: EnCodec(8 RVQ layers), SoundStream, DAC 등 다층 RVQ codec, SpeechTokenizer(semantic + acoustic 분리).
- 제안 방법: 단일 코드북(single quantizer)만으로 24kHz 오디오 1초를 40 또는 75개 토큰으로 압축하는 WavTokenizer. (1) 확장된 VQ 공간(broader VQ space) + K-means 초기화 + random awakening으로 코드북 활용률 극대화, (2) extended contextual window와 개선된 attention, (3) multi-scale discriminator + inverse FFT 구조로 고품질 재구성.
- 핵심 차별점: 극단적 압축(단일 코드북, 40/75 tokens/sec)에도 SOTA 재구성 품질(UTMOS 기준)을 달성하면서 richer semantic 정보를 자연스럽게 보존. 다층 RVQ 대비 AR 모델링 시퀀스 길이를 수 배 단축.
- 실험 결과: Speech, audio, music 3개 도메인에서 SOTA UTMOS 점수 달성. 단일 코드북으로 기존 다층 RVQ 대비 동등 이상의 재구성 품질을 기록.
- 우리 연구 관련도: ★★★ (단일 VQ 코드북 설계, VQ 공간 확장 전략, semantic 정보 보존이 VQ-based speech representation 연구와 매우 밀접. Prediction-free SSL의 이산 표현 설계에 직접 참고 가능)

### **k2SSL: A Faster and Better Framework for Self-Supervised Speech Representation Learning (Yifan Yang, Zengrui Jin, Ziyang Ma et al. — Shanghai Jiao Tong University, Tsinghua University, 2024-11)** — arxiv preprint
- arxiv URL: https://arxiv.org/abs/2411.17100
- 기존 문제: Transformer/Conformer 기반 SSL backbone(HuBERT, WavLM 등)이 주류이지만 ASR에서 뛰어난 Zipformer 같은 encoder는 SSL에서 탐색되지 않음. 기존 SSL 프레임워크(fairseq)는 대규모 데이터 처리 시 비효율적이고 메모리 사용량이 높음.
- 기존 방법: fairseq 기반 HuBERT/WavLM 학습, Conformer backbone SSL, BEST-RQ 등 대안적 SSL.
- 제안 방법: Zipformer encoder를 SSL backbone으로 도입한 k2SSL 프레임워크. HuBERT 아키텍처에서 성능 기여도가 낮은 memory-intensive 컴포넌트를 제거하고, ScaledAdam 옵티마이저를 통합하여 학습 효율과 성능을 동시에 개선. 오픈소스로 공개.
- 핵심 차별점: Zipformer 기반 SSL이 HuBERT/WavLM 대비 파라미터 효율·학습 속도·WER 모두에서 우위. 8×V100으로 Base 모델 사전학습이 가능한 접근성.
- 실험 결과: LibriSpeech에서 Zipformer Base가 HuBERT Base 대비 fine-tuning 후 최대 34.8% 상대 WER 감소, GPU 시간 기준 3.5배 사전학습 속도 향상. 60K 시간 LibriLight로 스케일업 시 Zipformer Large가 HuBERT Large 성능을 5/8 사전학습 스텝만으로 달성.
- 우리 연구 관련도: ★★☆ (prediction-based SSL(HuBERT 계열)의 효율적 대안으로, prediction-free SSL 연구의 직접 비교군이자 backbone 설계 참고. VQ 기반 표현과의 직접 관련은 moderate하지만 SSL 프레임워크 설계 관점에서 유용)

## 이번 주 트렌드 요약 (2026-04-28 ~ 2026-05-01)

1. **Diarization + ASR 통합의 다양화**: Sortformer(Sort Loss 기반 permutation 해결)와 DM-ASR(다화자 ASR을 multi-turn dialogue로 재정의) 등 diarization과 ASR을 통합하는 접근법이 다양화되고 있음. 특히 LLM을 ASR 과제에 직접 활용하는 흐름이 가속화.

2. **Voice Agent의 Agentic 전환**: Audio2Tool(tool calling 벤치마크)과 VoxMind(end-to-end agentic dialogue)에서 볼 수 있듯, SpeechLLM이 단순 대화를 넘어 도구 호출·외부 지식 연동 등 agentic 능력을 갖추는 방향으로 진화 중. 그러나 복합 의도와 음향 도전 환경에서는 여전히 성능 저하가 큼.

3. **Speech Tokenizer의 대규모 스케일링**: SiTok(1.6B 파라미터, 22M 시간 학습)과 WavTokenizer(단일 코드북 극한 압축)에서 볼 수 있듯, speech tokenizer가 더 적은 토큰으로 더 많은 semantic + acoustic 정보를 담는 방향으로 스케일링되는 추세. Diffusion decoder 도입(SiTok)이 새로운 디코딩 패러다임으로 부상.

4. **SSL 프레임워크의 효율화 경쟁**: k2SSL(Zipformer backbone, 3.5배 속도 향상)처럼 SSL 사전학습의 접근성과 효율성을 높이는 연구가 지속. 이는 VQ 기반/prediction-free SSL 연구에서도 효율적 backbone 선택의 중요성을 시사.

5. **모듈식 SpeechLLM 생태계 성숙**: SLAM-LLM(IEEE JSTSP accepted)처럼 encoder-projector-LLM을 자유롭게 조합할 수 있는 모듈식 프레임워크가 학술적으로 인정받기 시작. SSL 표현의 downstream 활용 실험 인프라로서 가치가 커지고 있음.
